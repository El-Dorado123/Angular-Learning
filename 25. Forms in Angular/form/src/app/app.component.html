Forms Types in Angular

1. Template - driven Form

2. Reactive form

Template Driven Forms

  • Based on HTML DOM and Angular
  • use to create simple forms
  • limited controls over form Element
  • Normal Validation

Reactive Forms

  • use to create Complex forms
  • Based on class Body programming
  • Full Control on form Elements
  • Sync with HTML

    • Template --> class --> service --> database
