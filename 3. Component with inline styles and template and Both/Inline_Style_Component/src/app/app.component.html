Component With Inline Style And Template

Generate component with inline template and style using Angular CLI. But when we want to generate component with inline template and style we have to provide two options after the above command. For inline templates, we need to add --inlineTemplate and For inline styles, we need to add --inline-style and For Both , we need to add --inline-template --inline-template

Inline style Component :- it contains html,spec.ts,ts file and we give style using style property of decorator of component.ts file.

Inline Template Component :- it contains css,spec.ts,ts file and we write html in template property of decorator of component.ts file.

Inline style Component :- it contains spec.ts,ts file and we use style and template property of decorator of component.ts file.
