Automation Testing (Unit Testing)

Automation Testing testing

Automated testing is a software testing technique that automates the process of validating the functionality of software and ensures it meets requirements before being released into production.

Types of Automation Testing

  • Unit Test Case
  • Integration
  • End-to-End

There are two main types of Angular testing: unit testing and end-to-end testing. Unit testing is verifying that individual units of code behave as expected. End-to-end testing is verifying that the entire application behaves as expected.

Angular Automation Testing

Protractor - Run Test Case on Browser.

Jasmine - Testing Environment,Run spec file for the test case.

Karma - Run test case in multiple browser.
