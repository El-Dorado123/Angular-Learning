InterPolation

In AngularJS, Interpolation is a way to transfer the data from a TypeScript code to an HTML template

How can we use it?

we can use it using { { } }

What we can do with Interpolation?

{{title}}

Addition : 2+2

Addition : {{2+2}}

Substraction : 2-2

Substraction : {{4-2}}

Multiply : 2*2

Multiply : {{2*2}}

Comparision : title == 'InterPolation'

Comparision : '{{title=='Interpolation'}}'

What we can't do with Interpolation?

Assignment : { { title='Hello World' } }

Typeof : { { typeof title } }

Increment : { { number++ } }

Decrement : { { number-- } }
